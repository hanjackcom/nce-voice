# nce-voice

这是一个存放新概念英语音频文件的仓库，包含原始音频文件和经过去片头片尾的音频文件和合成单一文件的音频文件。仓库致力为学好英语的伙伴们提供上班时也能摸鱼学英语的机会！Oh, My God!

---

### 📚 上班摸鱼版新概念英语听力

- 去掉每个音频文件片头课程号播报语音和片尾铃声。

- 把每册所有语音拼接成一个单一文件，方便直接播放，不用按文件打开。

- 享受沉浸式听力，不再被冗余声音打扰。

- 目录分门别类，方便按需查找使用。
